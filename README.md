# Study Notes

Study Notes is a simple study tool for building themes, subthemes, notes, quotes, and source references in one calm environment.

## Vision

The app is designed for study rather than technical users. It helps users:

- build main themes and subthemes
- store notes, quotes, and source references
- create meaningful links between study items
- search and rediscover earlier material
- work individually or in shared study spaces

## Core Principles

- simple and calm interface
- content first, structure second
- desktop-first and responsive
- list views as the main experience
- meaningful links with short explanations
- easy collaboration with clear permissions

## Planned MVP

The first version focuses on:

- personal and shared workspaces
- main themes and subthemes
- three item types: note, quote, source reference
- simple linking between items
- search across study content
- workspace roles: admin, writer, reader
- hidden global admin for recovery and oversight

The application uses Next.js, TypeScript, and plain CSS.

```bash

npm run dev
```

Open `http://localhost:3000` to view the application.
